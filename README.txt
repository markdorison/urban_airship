# Urban_Airship Readme
This module allows site administrators to use a Drupal site to send push
notifications to the [Urban Airship](http://urbanairship.com/) service.

## What can I do with this module?
*   Send push notifications to mobile devices via Urban Airship Display a Views
*   listing of all previously-sent push notifications Delete previously-sent
*   push notifications (through VBO integration)

## Requirements
*   Views Views Bulk Operations

## Background This module was initially developed for the [New
Gov](http://itunes.apple.com/us/app/new-gov/id475150622?mt=8) iPhone app,
developed by the [Mobile Technology Center](http://mobiletech.apsu.edu/) at
[Austin Peay State University](http://www.apsu.edu/).
